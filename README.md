# Expirix 🚀
Smart Expiry Tracker for groceries & medicines.

## Features
- Track expiry dates of groceries & medicines
- Scan barcodes & read expiry from images using AI
- Get reminders before things expire
- Donate items nearing expiry to NGOs

## Stack
- React.js frontend
- Node.js + Express backend
- MongoDB database
- TensorFlow.js + OpenCV for image processing
